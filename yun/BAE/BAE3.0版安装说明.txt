1.修改zb_users/c_option.php文件

a)使用MySQL数据库

'ZC_MYSQL_USERNAME' => '填入API Key'  里填写API Key
'ZC_MYSQL_PASSWORD' => '填入Secret Key'  里填写API Key
'ZC_MYSQL_NAME' => '填入数据库名称'  里填写MySQL数据库名称
保存即可


b)使用SQLite3数据库
将'ZC_DATABASE_TYPE' => 'mysql'  改为 'ZC_DATABASE_TYPE' => '' 保存即可



2.通过用SVN上传到BAE，并点“快捷发布”。



3.打开应用的二级域名或是绑定域名，进入安装界面，

设置“网站名”“用户名”，“密码”等即可，无需改动数据库设置（使用SQLite3数据库时需要确定选项）。


4.ZBP的BAE版已包含有app.conf，支持伪静态。
